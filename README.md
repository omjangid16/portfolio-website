# Portfolio Website

A personal portfolio website built with React and Vite.

## Prerequisites

- [Node.js](https://nodejs.org/) (Version 16 or higher recommended)

## Setup Instructions

1.  **Open a Terminal**: Navigate to the project folder where this file is located.
2.  **Install Dependencies**: Run the following command to download the necessary libraries:
    ```bash
    npm install
    ```

## Running the Project

1.  **Start Development Server**: To view the website locally, run:
    ```bash
    npm run dev
    ```
2.  **Open Browser**: The terminal will show a local URL (usually `http://localhost:5173`). Ctrl+Click it or copy-paste it into your browser.

## Building for Production

To create an optimized build for deployment (e.g., to Vercel or Netlify):
```bash
npm run build
```
The output will be in the `dist` folder.
